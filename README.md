# CarDekho Data Analysis Using MySQL

## Project Overview
This project analyzes the CarDekho dataset using MySQL to extract meaningful business insights related to car pricing, fuel types, ownership trends, and vehicle performance.

## Tools Used
- MySQL
- SQL Queries
- Data Analysis

## SQL Concepts Used
- SELECT Statements
- WHERE Clause
- GROUP BY
- ORDER BY
- Aggregate Functions
- Subqueries

## Objectives
- Analyze used car market trends
- Compare selling prices across fuel types
- Understand ownership distribution
- Identify key factors affecting resale value

## Files Included

| File Name | Description |
|---|---|
| cardekho_dataset.csv | Dataset used for analysis |
| cardekho_dataset_analysis.sql | SQL queries used for business analysis |

## Key Insights
- Petrol cars dominate the dataset
- Selling prices vary significantly based on car age
- Ownership history impacts resale value
- Fuel type and transmission influence pricing trends
